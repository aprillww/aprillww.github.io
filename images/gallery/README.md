# 作品展示图片说明

## 用途
此目录用于存放各种作品展示图片，包括培训成果、纪念册样本、视频截图等

## 目录结构建议
```
gallery/
├── courses/          # 课程相关图片
│   ├── business-english/
│   ├── ielts/
│   └── trade-skills/
├── memory-books/     # 纪念册作品展示
├── videos/          # 视频作品截图
├── certificates/    # 证书和荣誉
└── testimonials/    # 学员反馈图片
```

## 推荐规格
- 尺寸: 建议统一比例（如16:9或4:3）
- 格式: JPG或PNG
- 大小: 每张图片控制在300KB以内
- 质量: 高清晰度，能清楚展示作品细节

## 使用建议
- 添加水印保护版权
- 准备缩略图和大图两个版本
- 按类别分文件夹管理
- 定期更新最新作品

## 展示方式
可以通过JavaScript创建图片画廊或轮播图展示
